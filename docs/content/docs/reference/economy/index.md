# Economy Reference

Use this track when you are integrating player-facing gameplay rules rather than just content or UI.

These pages explain how inventory, trade, and quests behave as runtime systems once the player can own items, buy and sell, and progress through objectives.

- slot-based item ownership
- trader stock and pricing rules
- buy/sell/gift transaction semantics
- quest lifecycle, prerequisites, and event-driven progress
